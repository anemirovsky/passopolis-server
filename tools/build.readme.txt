To install:

$ cd ~/Downloads
$ wget https://hydra.nixos.org/build/25489757/download/1/nix_1.10-1_amd64.deb
$ sudo dpkg -i nix_1.10-1_amd64.deb

Before trying to compile mitro:
$ source /etc/profile.d/nix.sh
$ nix-channel --update

See https://github.com/anemirovsky/passopolis-server for build commands.

* https://www.hashtagsecurity.com/mitro-login-manager-on-premise-2/ for how to run it on the server for reals.
* https://github.com/mitro-co/mitro/issues/123#issuecomment-128889912
* https://github.com/mitro-co/mitro/blob/master/mitro-core/production/ansible/README.md

Debugging:
$ java -ea -jar -agentlib:jdwp=transport=dt_socket,address=9999,server=y,suspend=n -Ddatabase_url='jdbc:postgresql://localhost:5432/DATABASE?user=DATABASE_USER&password=DATABASE_PASSWORD' -DgenerateSecretsForTest=true -Dhttps_port=9443 build/mitrocore.jar
$ jdb -attach 9999 -sourcepath build/java/src/

See https://docs.oracle.com/javase/8/docs/technotes/tools/windows/jdb.html
